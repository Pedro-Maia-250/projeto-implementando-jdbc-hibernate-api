# Backend Architecture Study

Este projeto demonstra diferentes estratégias de persistência de dados
implementadas sobre a mesma base de domínio.

## Implementações disponíveis

- Branch jdbc-version → DAO + JDBC puro
- Branch hibernate-version → JPA/Hibernate
- Branch springboot-version → API REST com Spring Boot

Cada branch representa uma abordagem arquitetural distinta.
